# Nifty100 Financial Intelligence Platform

## Project Overview

A financial analytics platform built using Python, SQLite, Pandas, and Streamlit to analyze Nifty100 companies.

## Features

* ETL Pipeline for financial datasets
* SQLite Database Integration
* Data Quality Validation
* Financial Ratio Engine
* Growth Analytics
* Investment Screener
* Interactive Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* SQLite
* Streamlit
* GitHub

## Dashboard Modules

* Company Analysis
* Investment Screener
* Top Companies Ranking
* Growth Analytics
* Database Statistics

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run dashboard/app.py
```
